# 作品集模块规范

## 概述

作品集模块是个人作品展示网站的核心功能区域，用于系统化展示工程师过往的项目经验和专业技术能力。该模块通过卡片网格布局呈现项目作品，支持按技术分类筛选查看，点击项目卡片可展开查看详细信息。作品集模块的设计目标是让潜在客户在 2 至 3 分钟内快速了解工程师的项目经验深度和技术栈广度，从而建立专业信任并产生合作意向。

每个项目作品需要展示足够的技术细节，包括项目背景、技术方案、核心功能、开发周期和客户评价等信息。模块采用无限滚动或分页加载方式展示项目列表，避免单页加载过多内容影响性能。项目卡片设计应兼顾信息密度和视觉简洁，在有限空间内传达项目亮点和技术价值。

## 功能需求

### 项目卡片

项目卡片是作品集模块的基本展示单元，采用垂直排列的卡片结构。卡片顶部显示项目封面图片，尺寸比例 16:9 或 4:3，图片加载失败时显示占位背景色。图片下方为项目标题和一句话描述，标题使用 18px 加粗字体，描述使用 14px 灰色文字。标题下方以标签形式显示项目使用的技术栈，每个标签对应一种技术（如 Go、MySQL、Redis），点击标签可筛选同类型项目。

卡片底部显示项目的关键指标或核心成果，如「日活 50 万」「响应时间 < 50ms」「QPS 10 万+」等数据。卡片右下角显示「查看详情」链接或按钮，悬停时展开显示更多信息预览。项目卡片支持三种尺寸规格：大卡片（桌面端三列布局）、中卡片（平板端两列布局）、小卡片（移动端单列布局），根据视口宽度自动调整。

### 分类筛选

分类筛选栏位于作品集模块顶部，包含全部、技术分类、角色分类三个维度。技术分类按项目使用的主要技术划分，包括「后端开发」「微服务」「API 接口」「分布式系统」「性能优化」等类别。角色分类按项目中的职责划分，包括「技术负责人」「核心开发」「架构设计」「技术顾问」等。筛选栏采用可多选的胶囊标签样式，点击标签切换选中状态，支持组合筛选。筛选结果实时更新，配合加载动画显示筛选进度。重置按钮可一键清除所有筛选条件。

### 项目详情

点击项目卡片或「查看详情」按钮后，以模态框形式展开项目详情页面。模态框覆盖整个视口，背景使用半透明黑色遮罩。模态框顶部显示项目标题和封面大图，图片高度不超过视口高度的 40%。详情页面分为五个内容区块：项目概述、技术方案、核心功能、开发数据和客户评价。

项目概述包含项目背景、解决的问题、业务价值和项目周期（开始时间至结束时间）。技术方案详细描述系统架构、技术选型、核心设计模式和关键实现思路，配合架构图或流程图辅助说明。核心功能列举项目的主要功能模块，每个功能有简短描述和使用的技术点。开发数据展示项目的技术指标，如并发量、响应时间、存储容量、代码规模等量化信息。客户评价引用合作方的真实反馈，包含评价内容和客户名称（或公司名称）。

模态框右上角显示关闭按钮，点击关闭按钮、点击遮罩区域或按 ESC 键均可关闭详情页。模态框内部支持滚动，滚动时顶部固定显示项目标题。

### 加载与分页

作品列表初始加载时显示骨架屏动画，模拟卡片结构占位。数据加载完成后骨架屏淡出，实际内容淡入。加载失败时显示重试按钮和错误提示。作品列表采用无限滚动方式加载更多，点击「加载更多」按钮或滚动到底部自动触发下一页加载。底部加载状态显示加载动画或「没有更多内容」提示。每个项目卡片包含在列表中和在详情页中的两种渲染模式，根据显示场景优化布局和信息密度。

## 界面需求

### 视觉规范

作品集模块整体采用与首页一致的深色主题。卡片背景色使用 #1e293b（深灰蓝），圆角 12 像素，边框 1 像素 #334155。卡片悬停时背景色变为 #334156，边框颜色变浅，添加上浮阴影效果。技术标签使用不同颜色区分技术类别：后端相关使用蓝色、微服务相关使用紫色、性能相关使用橙色、架构相关使用青色。标签文字颜色与背景形成高对比度，确保可读性。

模态框背景色使用 #0f172a，宽度在桌面端限制为 800px 至 1000px，两侧留白对称。模态框内部区块之间使用 24px 垂直间距，文字行高 1.6。标题与正文使用不同字重和颜色区分层次。架构图和流程图使用 SVG 矢量格式，在任意分辨率下保持清晰。

### 响应式设计

桌面端（屏幕宽度 ≥ 1024px）显示三列项目卡片，每行三个，卡片宽度自适应。筛选栏横向展开显示所有分类标签。平板端（屏幕宽度 768px 至 1023px）显示两列项目卡片，筛选栏部分标签折叠为「更多」下拉菜单。移动端（屏幕宽度 < 768px）显示单列项目卡片，筛选栏水平滚动显示，模态框占据全屏。触摸设备的卡片悬停效果改为点击展开预览。

### 动画效果

项目卡片加载时采用交错淡入动画，每张卡片延迟 100 毫秒依次出现。筛选切换时卡片重新排列，使用 300 毫秒的布局过渡动画。打开模态框时卡片图片放大并淡出背景，模态框内容从底部滑入并淡入。关闭模态框时反向播放进入动画。详情页内部滚动流畅无卡顿，锚点跳转平滑滚动。

分类元数据存储于 data/categories.json，定义技术分类和角色分类的显示名称、颜色编码和排序权重。

## 验收标准

1. 桌面端显示三列卡片、平板端两列卡片、移动端单列卡片，布局正确无溢出。
2. 分类筛选支持单选和多选，筛选结果实时更新，与项目数据准确匹配。
3. 项目详情模态框在各种屏幕尺寸下正确显示，内容不截断，滚动功能正常。
4. 页面 Lighthouse 性能评分：性能 ≥ 85（因图片较多可适当放宽），可访问性 ≥ 90。
5. 项目图片懒加载，首屏加载时间不超过 3 秒。
6. 所有交互元素有点击反馈，动画流畅不卡顿。
7. 数据文件格式校验通过，缺失字段有合理的默认值或显示「待补充」。
8. 项目详情页的 SEO 结构化数据正确生成，支持社交媒体预览卡片。
